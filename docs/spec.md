# lucky-wheel 抽獎轉盤 — spec

2026-07-16。行銷工具箱替換計畫第 2 案,替換對象:Wheel of Names。

## 定位

台灣小編的抽獎轉盤:貼上名單(或直接貼 FB 留言串一鍵清洗),全螢幕轉盤抽獎,中獎名單一鍵複製。純前端、無廣告、零追蹤。

## 比原版好在哪

- **FB 留言一鍵洗名單**:貼上從 FB 留言區複製的整串文字,自動去掉「讚/回覆/分享/N天前」等雜訊與重複,留下名字(可手動修)——原版要自己整理
- 繁中介面、無廣告(原版有橫幅廣告)
- 中獎名單面板:抽幾位都記著,一鍵複製去公告(原版 Results 藏在側欄)
- 公平性:winner 用 crypto.getRandomValues 決定,轉盤只是演出
- 零追蹤、名單不離開瀏覽器(原版名單存它的雲端)

## 功能

1. 名單編輯:textarea 一行一名 + 計數 + 去重 + 洗牌。
2. FB 留言清洗:摺疊區貼原始留言文字 → 過濾固定雜訊詞(讚、回覆、分享、編輯、作者、置頂、追蹤、檢舉、查看更多、翻譯、N 則回覆)、時間樣式(3天、5小時、2w…)、純數字行、過長行(>25 字視為留言內容)、連續重複 → 填回名單。
3. 轉盤:canvas 繪製,5 色循環;crypto 選 winner → 6 圈 ease-out 轉到該格;中獎彈窗 + 彩帶粒子;「中獎後自動移除」開關。
4. 中獎名單:依序記錄、一鍵複製、清空。
5. 全螢幕按鈕(直播用)。
6. localStorage 自動保存名單與中獎名單。
7. 測試鉤子:`?fast=1` 縮短轉動時間(驗證腳本用)。

單檔 index.html、深淺色、RWD、OG/favicon、footer 三件套+回連行銷工具箱、MIT 林亞澤、正體中文、無 emoji 文案。
